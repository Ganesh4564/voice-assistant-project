# Conversational Voice Assistant

## Project Overview
This project is a browser-based conversational voice assistant. It interacts with the user through speech, asks a simple question, listens to the user response, and performs a Google search based on the spoken request.

## Features
- Full-screen modern user interface
- Voice interaction
- Speech-to-text using Web Speech API
- Text-to-speech assistant response
- Conversational flow
- Automatic Google search

## Tools and Technologies
- HTML
- CSS
- JavaScript
- Web Speech API
- Google Chrome

## How to Run
1. Download or clone this repository.
2. Open `voice_assistant.html` in Google Chrome.
3. Allow microphone permission.
4. Click **Start Assistant**.
5. Speak your response and search topic.

## Example Flow
Assistant: Hi, how are you doing today?  
User: I am good.  
Assistant: Great! What do you want me to search?  
User: Artificial intelligence in healthcare.  
System: Opens Google search results.
